Odoo Woocommerce Connect

This Module will Connect Odoo with Wordpress WooCommerce and synchronise Data.

Documentation

Homepage

  • The home page has a list of connections made to the Wordpress Website
  • For making the connection following parameters are required
    • URL of the Website
    • Consumer Key
    • Consumer Secret
    • Version

Product Template Mapping

  • Product Template Mapping gives a detailed description of products along with its various varients availabe
  • Product Templates contains the overall information of the product along with its different attributes and categories

Product Attribute

  • Product Attribute Mapping gives a list of products which can be used to differentiate between the products based of different attributes
  • It Contains Attribute name, Backend name and Woo-Id

Product Product Mapping

  • Product Template Mapping gives a list of products available in the System without any redundant products
  • It Contains Attribute Name ,Backend Name ,Woo-Id and Image of the product

Product Attribute Value

  • Product Attribute Value gives a list of products which can be used to differentiate between the products based of different values of various Attributes

Category Mapping

  • Product Category gives a list of various categories of the products
  • The Category can be based on physical appearance,services,memory of the electronic product.etc

Customer Mapping

  • Customer Mapping gives a list Available Customers in the System

Invoice Refund Mapping

  • Customer gets the Invoice refund only when the balance amount completely is paid
  • Refund Invoice is generated only after refund amount is paid
